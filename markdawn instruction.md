## Списки.

Чтобы добавить не нумерованные списки, необходимо нужные пункты выделить звезочкой (*). Например вот так:

* Элкмкент 1
* Элемент 2 
* Элемент n

Чтобы добавить нумерованные списки, необходимо их просто пронумировать. например вот так:
1. Список
2. Список 
3. Список
