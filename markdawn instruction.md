# Инструкция для работы с Markdown.

## Выделение текста.
Чтобы выделить текст курсивом, необхадимо обрамить его звездочками * ( *текст* )

Чтобы выделить текст полужирным, необходимо обрамить его двойными звездочками ** ( **текст** )
